DB 테이블 사용설명서

축약어 정리

PK: Primary Key

FK: Foreign Key

AI: AUTO_INCREMENT

NN: Not Null

admin ( 관리자 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK adminIdx int 관리자 인덱스 AI 고유 식별자, 자동 증가
adminId varchar(20) 관리자 아이디 NN
adminPassWord varchar(20) 관리자 비밀번호 NN
adminGrade varchar(20) 관리자 등급 NN main: 주매니저, sub: 부매니저

user ( 사용자 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK userIdx bigint 사용자 인덱스 AI 고유 식별자, 자동 증가
userContact varchar(20) 사용자 연락처
userJoinDate datetime 사용자 등록일 now()
userGrade varchar(20) 사용자 등급 Silver, Gold, Platinum, Diamond.다이아가 제일 높음.

userLog ( 사용자 기록 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK logIdx int 사용자 아이디 AI 고유 식별자, 자동 증가
userIdx bigint 회원 인덱스 NN 사용자 테이블의 외래 키
logTop1ProductIdx bigint top1메뉴 인덱스 null 상품 테이블의 외래 키
logTop2ProductIdx bigint top2메뉴 인덱스 null 상품 테이블의 외래 키
logTop3ProductIdx bigint top3메뉴 인덱스 null 상품 테이블의 외래 키
logDate date 기록 날짜 date
logGrade varchar(20) 기록 등급
logOrderCount smallint 기록 주문횟수 0
logExpense int 기록 소비금액 0
logCouponUsage tinyint 기록 쿠폰사용수 0

couponSetting ( 쿠폰세팅 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK couponSettingIdx bigint 쿠폰세팅 인덱스 AI 고유 식별자, 자동 증가
FK productIdx bigint 상품 인덱스 NN 상품 테이블의 외래 키
couponSettingGrade varchar(10) 쿠폰세팅 등급 NN Silver, Gold, Platinum, Diamon. 다이아가 제일 높음
couponSettingRate int 쿠폰세팅 %할인율 0
couponSettingFixed int 쿠폰세팅 고정할인금액 0
couponSettingStatus boolean 쿠폰세팅 사용상태 false 매월 초에 false: 쿠폰 지급 안함, true: 쿠폰 지급 함

coupon ( 쿠폰 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK couponIdx bigint 쿠폰 인덱스 AI 고유 식별자, 자동 증가
FK userIdx bigint 사용자 인덱스 NN 사용자 테이블의 외래 키
couponName varchar(100) 쿠폰 이름
couponRate int 쿠폰 %할인율 0
couponFixed int 쿠폰 고정할인금액 0
couponRegDate datetime 쿠폰 발급일 NN
couponExpDate datetime 쿠폰 만료일 NN
couponStatus boolean 쿠폰 사용상태 false false: 쿠폰 사용 안함, true: 쿠폰 사용 완료

product ( 상품 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK productIdx bigint 상품 인덱스 AI 고유 식별자, 자동 증가
productName varchar(1000) 상품 이름
productPrice smallint 상품 가격 0
productCategory tinyint 상품 카테고리 1: 단품, 2: 세트, 3:스낵, 4: 음료
productCalories smallint 상품 칼로리 0
productRecommendStatus boolean 추천상품여부 false true: 추천상품, false: 일반상품
productSaleStatus boolean 판매상태 true true: 판매중, false: 미판매
productRegDate datetime 상품 등록일 now()

option ( 옵션 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK optionIdx bigint 옵션 인덱스 AI 고유 식별자, 자동 증가
optionName varchar(50) 옵션 이름
optionPrice smallint 옵션 가격 0
optionCategory tinyint 옵션 카테고리 1 1: 스낵, 2: 음료
optionCalorie smallint 옵션 칼로리 0
optionSaleStatus tinyint 옵션 판매상태 true true: 판매중, false: 미판매

order ( 주문 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK oderIdx bigint 주문 인덱스 AI 고유 식별자, 자동 증가
FK1 userIdx bigint 사용자 인덱스 NN 사용자 테이블의 외래 키
FK2 productIdx bigint 상품 인덱스 NN 상품 테이블의 외래 키
FK3 couponIdx bigint 쿠폰 인덱스 null 쿠폰 테이블의 외래 키, 쿠폰 사용시 입력
FK4 option1Idx bigint 옵션1 인덱스 null 옵션 테이블의 외래 키, 세트 메뉴시 입력
FK5 option2Idx bigint 옵션2 인덱스 null 옵션 테이블의 외래 키, 세트 메뉴시 입력
orderNumber mediumInt 주문번호 0 주문번호, 매일 갱신, 멤버 변수에서 값 받음
orderProductCategory smallint 상품 카테고리 1 상품 카테고리 1: 단품, 2: 세트, 3: 스낵, 4: 음료
orderCount tinyint 주문수량 0
orderStatus boolean 조리상태 false false: 조리중, true: 조리완료
orderCalorie smallint 칼로리 0
orderDate datetime 주문일자 now()
orderPrice smallint 주문가격 0
