DB 테이블 사용설명서

축약어 정리

PK: Primary Key

FK: Foreign Key

AI: AUTO_INCREMENT

NN: Not Null

admin ( 관리자 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK adminIdx int 관리자 인덱스 AI 고유 식별자, 자동 증가
adminId varchar(20) 관리자 아이디 NN
adminPassWord varchar(20) 관리자 비밀번호 NN
adminGrade varchar(20) 관리자 등급 NN main: 주매니저, sub: 부매니저

user ( 사용자 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK userIdx bigint 사용자 인덱스 AI 고유 식별자, 자동 증가
userContact varchar(20) 사용자 연락처
userJoinDate datetime 사용자 등록일 now()
userGrade varchar(20) 사용자 등급 Silver, Gold, Platinum, Diamond.다이아가 제일 높음.

order ( 주문 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK oderIdx bigint 주문 인덱스 AI 고유 식별자, 자동 증가
FK1 userIdx bigint 사용자 인덱스 NN 사용자 테이블의 외래 키
FK2 productIdx bigint 상품 인덱스 NN 상품 테이블의 외래 키
FK3 couponIdx bigint 쿠폰 인덱스 쿠폰 테이블의 외래 키, 쿠폰 사용시 입력
orderNumber int 주문 번호 0 당일 주문번호, 다음날 초기화, 멤버변수 사용
orderAmount int 주문 품목수량 0
orderStatus boolean 주문 조리상태 false true: 조리완료, false: 조리중
orderDate datetime 주문 일자 now()
orderTotalPrice int 주문 총가격 0

쿠폰세팅 ( couponSetting )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK couponSettingIdx bigint 쿠폰세팅 인덱스 AI 고유 식별자, 자동 증가
FK productIdx bigint 상품 인덱스 NN 상품 테이블의 외래 키
couponSettingGrade varchar(10) 쿠폰세팅 등급 NN Silver, Gold, Platinum, Diamon. 다이아가 제일 높음
couponSettingRate int 쿠폰세팅 %할인율 0
couponSettingFixed int 쿠폰세팅 고정할인금액 0
couponSettingStatus boolean 쿠폰세팅 사용상태 false 매월 초에 false: 쿠폰 지급 안함, true: 쿠폰 지급 함

coupon ( 쿠폰 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK couponIdx bigint 쿠폰 인덱스 AI 고유 식별자, 자동 증가
FK userIdx bigint 사용자 인덱스 NN 사용자 테이블의 외래 키
couponName varchar(100) 쿠폰 이름
couponRate int 쿠폰 %할인율 0
couponFixed int 쿠폰 고정할인금액 0
couponRegDate datetime 쿠폰 발급일 NN
couponExpDate datetime 쿠폰 만료일 NN
couponStatus boolean 쿠폰 사용상태 false false: 쿠폰 사용 안함, true: 쿠폰 사용 완료

product ( 상품 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK productIdx bigint 상품 인덱스 AI 고유 식별자, 자동 증가
productName varchar(1000) 상품 이름 NN
productPrice int 상품 가격 0
productStock int 상품 재고 0 단품/세트/사이드 메뉴 미적용, 옵션 메뉴에만 적용
productCategory varchar(50) 상품 카테고리 NN 버거, 사이드, 음료, 커피, 디저트 (추가 가능)
productCalories int 상품 칼로리 NN
productRegDate datetime 상품 입고일 NN 단품/세트/사이드 메뉴 미적용, 옵션 메뉴에만 적용
productExpDate datetime 상품 유통기한 NN 단품/세트/사이드 메뉴 미적용, 옵션 메뉴에만 적용
productStockCount int 상품 누적판매수 NN 옵션 메뉴 미적용, 단품/세트/사이드 메뉴 적용

recipe ( 레시피 )

키 컬럼명 데이터타입 컬럼한글명 기본값 설명
PK recipeIdx bigint 레시피 인덱스 AI 고유 식별자, 자동 증가
FK productIdx int 상품 인덱스 NN 상품 테이블의 외래 키
recipeSource varchar(100) 레시피(재료) 이름 NN
recipeAmout int 레시피(재료) 수량 0
recipeCalories int 레시피(재료) 칼로리 0
